Fluidic    rethinking the shell

Fluidic is part of the new generation of graphical consoles. It is in Alpha and should be considered very much experimental. As such, everything is subject to change. Using Fluidic to manipulate irreplaceable data is not recommended.

For a list of commands to try, enter suggest.
